Задача: В документе articles_*.json Мне нужно найти статьи, максимально подходящие по моей тематике и 
создать с ними markdown файл определённой структуры. 

1. Тематика: 

Я занимаюсь поверхностным легированием чугунных отливок в литейной форме. 
В полость литейной формы я засыпаю металлический порошок, либо окрашиваю стенки формы краской с содержанием металлического порошка, 
который имеет температуру плавления ниже температуры плавления чугуна 
(у чугуна температура плавления 1300-1400 градусов). 
После заливки металла в полость формы порошок расплавляется под действием температуры заливаемого металла и 
приплавляется к поверхности отливки, тем самым мы получаем поверхностно легированный слой на поверхности чугунной отливки.
Таким способом мы можем получать композиты и отливка с разными свойствами поверхности.

2. На основнаии файла articles.txt создай markdown документ с имененм: "suitable date_from - date_to" 
следующего типа:

2.1 
    1-я строчка файла - строка query из search_dict. 
    2-я строчка - дата поиска из search_dict: date_from - date_to
2.2
    В файле будет 3 раздела: 
    # Уровень 1 - Прямое совпадение с тематикой
    # Уровень 2 - Близкая тематика (обобщённая тематика, к которой подходят статьи на данном уровне)
    # Уровень 3 - Смежная тематика (обобщённая тематика, к которой подходят статьи на данном уровне)

    в каждом разделе будет список статей определённой структуры:
    
    Структура статей:
        ## id: - id статьи из файла articles.json
        - **title:** Заголовок статьи
        - **title_translate:** Перевод заголовка статьи
        - **abstract:** Abstract статьи
        - **abstract_translate:** Перевод abstract статьи
        - **authors:** - авторы
        - **link:** - ссылка на статью
        - **published:** - дата публикации
